Class: GameClient

GameClient

new GameClient(options)

GameClient is what a controller uses to talk with a game's GameServer.

Messages sent with sendCmd get sent to the game. Messages from the game are delivered to callbacks registered with addEventListener.

Parameters:
Name Type Description
options GameClient~Options

options.

Methods

addEventListener(eventType, listener)

Adds an event listener for the given event type. event types are made up by you. Sending a command from the game with

someNetPlayer.sendCmd('foo', {data: "bar"});

Will arrive at the event listener registered for 'foo' here and given an object {data: "bar"}.

Note: Currently only 1 listener is allowed per eventType. Adding a second listener for an specific eventType replaces the previous listener for that type.
