SyncedClock

Synced clock support

Methods

createClock(online, opt_syncRateSeconds, opt_callback) → {Clock}

Creates a clock, optionally synced across machines.

Parameters:
Name Type Argument Description
online boolean

true = synced, false = local
opt_syncRateSeconds number <nullable>

how often to sync clocks in seconds. Default is once every 10 seconds
opt_callback callback <nullable>

called the first time the clock is synchronized.
Returns:

the created clock
