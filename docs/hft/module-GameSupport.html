Module: GameSupport

GameSupport

Implements the common parts of HappyFunTimes games written in JavaScript.

It provides a render loop with frame count and game clock as well as handles pausing the game during development so as not to deplete the battries on your laptop. It can also show the framerate and display debugging info.
